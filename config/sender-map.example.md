# Sender map example

Use one bullet per person. Format:

- Role | Display Name | email1@example.com / email2@example.com

Example:
- Professor | Ada Lovelace | ada@example.org
- Ph.D. Student | Grace Hopper | grace@example.org / g.hopper@example.org
- M.S. Student | Demo User | you@example.com

You can also add advisor-priority addresses in inline-code form if you want a quick bootstrap:
- Important contact: `advisor@example.org`
